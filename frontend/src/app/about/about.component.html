👥 About the Platform

A modern solution built to empower teams to work better, faster, and together.
Designed as a full-stack portfolio project and a real-world collaboration tool.

🔧 Key Features

📋 Kanban Boards

Organize tasks visually with drag & drop across custom columns.

📡 Real-time Updates

WebSocket-powered interactions ensure instant task & comment syncing.

💬 Comment System

Discuss tasks, leave notes, and keep everyone in the loop.

🔐 Role-based Access

Admins manage teams while members focus on getting things done.

🧱 Scalable Architecture

Built with maintainability, deployment, and future features in mind.

🛠 Tech Stack

  • 🟦 Angular + Standalone Components
  • 🐍 Django REST Framework (DRF)
  • 🧩 WebSocket server (Node.js / Django Channels)
  • 🔐 JWT Authentication
  • 🐳 Dockerized deployment
